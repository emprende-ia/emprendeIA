# EmprendeIA

EmprendeIA es una plataforma de incubación para emprendedores impulsada por Inteligencia Artificial. La aplicación ayuda a los usuarios a validar ideas de negocio, generar identidades de marca, crear planes de marketing personalizados y gestionar sus finanzas de forma intuitiva.

## Características Principales

- **Análisis de Viabilidad**: Evaluación crítica de ideas de negocio con semáforo de riesgo y FODA.
- **Identidad Digital**: Generación de nombre, eslogan, paleta de colores y logos con IA (Imagen 4.0).
- **Asistente Financiero**: Registro de transacciones y cálculo automático del punto de equilibrio.
- **Rutas de Aprendizaje**: Guías personalizadas de 14 días con soporte de audio TTS.
- **Buscador de Proveedores**: Recomendaciones basadas en ubicación y calidad del negocio.

## Stack Tecnológico

- **Frontend**: Next.js 15, React, Tailwind CSS, ShadCN UI.
- **Backend**: Firebase (Auth, Firestore, Storage, Cloud Functions, App Check).
- **IA**: Google Genkit con Gemini 2.5 Flash e Imagen 4.0.
